# ChromaDock

Minimalistic macOS color picker menu bar app built with SwiftUI.

## Quick Start

```bash
npm install
npm run dev        # Build and run
npm run installer  # Build release + create .dmg installer
```

## Features

- Menu bar droplet icon
- Color picker popover with thinMaterial background
- Copy hex code to clipboard
- Works as both menu bar app and Dock icon

## Requirements

- macOS 13.0+ (Ventura)
- Swift 6.0+ (comes with Xcode Command Line Tools)
